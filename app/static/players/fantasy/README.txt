Player images for BOWL Fantasy
==============================

This folder is used by:
- BOWL Fantasy (slug: bow)

Upload naming format (recommended):
  firstname_lastname_day_month_year.png

Example:
  connor_mcdavid_13_1_1997.png

Rules:
- Use lowercase letters.
- Use underscores between name/date parts.
- Day and month are not zero-padded.
- Preferred file type is .png.

Fallback behavior:
- If no match is found here, the app may still check app/static/players for legacy files.
