League History — trophy images (primary)

Place files under:  img/trophies/<league-slug>/
Example:         img/trophies/bowl-fantasy/hart_trophy.png

The award name is slugified for the filename stem (lowercase, non-alphanumeric -> underscores).
Example: "HART TROPHY" -> hart_trophy.png (or .webp / .jpg / .svg)

Known filename variants (also accepted):
- BOILEDEGG'S TROPHY -> boiledeggs_trophy.png (triple g) maps to the same card
- THE MASTERS' GREEN JACKET -> masters_green_jacket.png

If no file is found here, the app falls back to img/history/trophies/<league-slug>/.
