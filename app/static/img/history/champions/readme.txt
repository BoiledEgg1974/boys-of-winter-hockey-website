Championship banner images for the League History page.

Folder (repo root): app/static/img/history/champions/

Naming: Banner1.png, Banner2.png, … or banner1.png, banner2.png, … (any positive number; gaps are OK).

Optional space before the number is allowed (e.g. Banner 1.png). Extension .png is case-insensitive.

Use transparent backgrounds (no solid backdrop) so they sit cleanly on the dark page.
