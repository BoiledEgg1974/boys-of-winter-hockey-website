Championship banner images for the League History page.

Folder (repo root): app/static/img/history/champions/

Naming: Banner1.png, Banner2.png, … or banner1.png, banner2.png, … (any positive number; gaps are OK).

Optional space before the number is allowed (e.g. Banner 1.png). Extensions .png, .webp, .jpg, .jpeg are allowed (case-insensitive).

League-specific apps use a subfolder under this path (see HISTORY_CHAMPIONS_REL_DIR); the app also reads the root folder here for any banner index not present in the league folder.

Use transparent backgrounds (no solid backdrop) so they sit cleanly on the dark page.
