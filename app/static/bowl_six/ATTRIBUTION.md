# BOWL Six static assets

Decorative SVG artwork created for this project (no third-party image files).

- `rink-topdown.svg` — legacy top-down rink (unused on lineup)
- `rink-half-vertical.svg` — lineup half-rink (goalie in net, defense at blue line, forwards at center line); original artwork for this project, inspired by standard NHL rink diagrams
- `rink-hero.svg` — hub hero band
- `jersey-empty.svg` — empty jersey slot placeholder
